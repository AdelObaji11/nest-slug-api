# Nest Slug API

A simple NestJS application built as part of a learning project.

## Description

This project is a small API built with NestJS.

The application will include:

* CRUD operations
* Slug generation
* Integration tests
* Swagger documentation

## Project Setup

```bash
npm install
```

## Running the App

```bash
# development
npm run start:dev
```

The app runs on:

```txt
http://localhost:3000
```

## Running Tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e
